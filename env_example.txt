# Groq reasoning effort (only for reasoning models like gpt-oss)

# OpenAI model to use (only if MODEL_PROVIDER=openai)
# Options: gpt-4o-mini (default), gpt-4o, gpt-4-turbo, gpt-3.5-turbo
